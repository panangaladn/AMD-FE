## Getting Started

Follow these instructions to set up the project locally.

### Prerequisites
1. Node.js (v18+) ‚Üí [Download](https://nodejs.org/)  
2. npm or yarn ‚Üí Comes with Node.js / [Install Yarn](https://classic.yarnpkg.com/lang/en/docs/install/)  
3. Expo Go (for running on mobile) ‚Üí [Install on iOS/Android](https://expo.dev/client)

### Installation and Setup

1. Clone the repo
   ```bash
   git clone https://github.com/your-username/AMD-FE.git
   cd my-contact-app-frontend


2. Install dependencies

   ```bash
   npm install
   # or
   yarn install
   ```

3. Run the app (Expo)

   ```bash
   npm start
   # or
   yarn start
   ```

   Scan the QR code with the Expo Go app.
